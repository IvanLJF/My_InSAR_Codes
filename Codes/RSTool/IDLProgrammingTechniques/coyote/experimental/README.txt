Rename your current cgWindow.pro program cgWindow.old.pro. Then, install the contents of this zip file into your Coyote Library directory. The command "cgWindow" should work normally and you should be able to use it exactly as you were before.

If you have problems, please contact me:

David Fanning
david at idlcoyote dot com.

The following files are included.

   cgWindow -- Now a wrapper or skin for the cgCmdWindow object
   cgCmdWindow -- The heart and soul of the program. A widget object
                  that can be used as a smart draw widget object.
   cgWindowTest -- A test program that demonstrates one way to use
                  the new functionality.
